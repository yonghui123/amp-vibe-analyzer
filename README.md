# amp-vibe-analyzer

本项目为 **SignalX 旋转设备电流与振动信号采集分析系统** 的嵌入式固件工程，基于 **STM32F407VGT6** 开发。

## 快速入口

| 内容 | 路径 |
|------|------|
| 固件工程 | [`code_STM32/`](code_STM32/) |
| **编译 / ST-Link 烧录 / 接线 / 测试** | [`code_STM32/docs/项目说明文档.md`](code_STM32/docs/项目说明文档.md) |
| 产品需求 | [`code_STM32/docs/产品需求文档PRD.md`](code_STM32/docs/产品需求文档PRD.md) |
| PC 界面预览 | `code_STM32/pc_simulator` → `.\build_and_run.ps1` |

板端烧录与调试使用 **ST-Link（SWD）**，详见项目说明文档第四节。
