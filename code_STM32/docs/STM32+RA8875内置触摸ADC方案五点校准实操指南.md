## 1\. 校准核心原理（固定 6 参数仿射变换，工业标准五点校准）

RA8875 读出原始 ADC 值 \(Tx,Ty\) → 换算屏幕像素 \(Lx,Ly\)

```Plain Text
Lx = A*Tx + B*Ty + C
Ly = D*Tx + E*Ty + F
```

采集 5 组【屏幕目标像素 \+ 触摸原始 ADC】，算出 A/B/C/D/E/F，存 Flash，每次触摸实时换算。

## 3\. 完整代码流程（可直接移植）

### （1）读取 RA8875 内置触摸原始值

```c
// 读取10bit触摸ADC
void RA8875_ReadTP(uint16_t *tx, uint16_t *ty)
{
    uint16_t xh = RA8875_ReadReg(0x90);
    uint16_t xl = RA8875_ReadReg(0x91);
    uint16_t yh = RA8875_ReadReg(0x92);
    uint16_t yl = RA8875_ReadReg(0x93);
    *tx = (xh << 2) | xl;
    *ty = (yh << 2) | yl;
}

// 多次采样防抖，消除跳点
void TP_GetAverage(uint16_t *avgX, uint16_t *avgY)
{
    uint32_t sumX=0, sumY=0;
    for(uint8_t i=0; i<10; i++)
    {
        uint16_t x,y;
        RA8875_ReadTP(&x,&y);
        sumX += x;
        sumY += y;
    }
    *avgX = sumX / 10;
    *avgY = sumY / 10;
}
```

### （3）校准界面采集逻辑

1. 清屏，打印提示文字：请依次点击十字中心

2. 循环 5 次，依次绘制 5 个十字靶标

6. 将 tp\_cal 结构体存入 STM32 片内 Flash（带校验标志，防止垃圾数据）
