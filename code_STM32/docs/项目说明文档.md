# code_STM32 项目说明文档

## 一、项目概述

本项目为 **SignalX 旋转设备电流与振动信号采集分析系统** 的嵌入式固件工程，是 `code` 项目（Keil MDK）迁移到 **STM32CubeIDE** 的版本，使用 GCC 编译，**无代码大小限制**。

- **芯片**: STM32F407VGT6 (ARM Cortex-M4, 168MHz, 1MB Flash, 128KB RAM + 64KB CCMRAM)
- **显示屏**: 7寸 RA8875 触摸屏，分辨率 800×480
- **ADC**: AD7606（8通道同步采样，FSMC 接口）
- **RTOS**: FreeRTOS
- **GUI**: LVGL v8.4
- **IDE**: STM32CubeIDE 2.1.1，编译器 arm-none-eabi-gcc 14.3
- **调试器**: J-Link

## 四、编译烧录到板子

### 4.1 所需软件

| 软件 | 版本 | 说明 |
|------|------|------|
| STM32CubeIDE | 2.1.1+ | 含 arm-none-eabi-gcc |
| J-Link 驱动 | 最新 | Segger 官网 |
| GDB Server for J-Link | — | CubeIDE 内安装 |

### 4.2 打开与编译

1. **File** → **Open Projects from File System** → 选择 `e:\workspace\Signal_STM32\code_STM32`
2. 若 Windows 下 `sh.exe` Error 87：项目 **Properties** → **C/C++ Build** → **Environment** 添加 `SHELL=cmd.exe`
3. **Project** → **Build Project**（Ctrl+B）

### 4.3 烧录与调试

Run/Debug 配置选用 J-Link，SWD 连接板子后 Download / Debug。

## 六、采集与主监控模块（acq，2026-07）

八通道 AD7606 采集、校准路由、主监控实时曲线。详见 `docs/八通道采集与主监控显示工作任务.md`。

| 模块 | 路径 | 职责 |
|------|------|------|
| BSP | `user/bsp/bsp_ad7606.*` | 8 路 raw、1kHz、`AD7606_SampleHook` |
| 管线 | `acq_pipeline.c` | 校准、通用路由、振动环 512 点、PP、包络 |
| 任务 | `acq_task.c` | AcqTask **20ms**、`Acq_Start/Stop` |
| 报警 | `alarm_manager.c` | 阶段 8 前占位，恒 Normal |
| GUI | `GUI_PostRealtimeData` | AcqTask → `lv_async_call` → chart |

**线程模型**：`guiTask` → `lv_timer_handler`；`acqTask` **不**直接操作 `lv_chart`。

**GUI 内存相关（已实现）**：

- Tab **懒加载**（`gui_load_tab`）
- 非 Main Tab：`gui_apply_realtime_payload` 跳过 chart
- **待优化**：非 Main 时仍 50Hz `lv_mem_alloc`+async → 见 `LVGL内存与刷新优化工作计划.md` 任务 3.6

## 九、注意事项

1. 不要手改 `Core/`、`Drivers/`（CubeMX 再生）
2. 业务代码修改后同步到 `code/user/`（`inlcude` ↔ `include`）
3. 板端 LVGL：`Middlewares/Third_Party/LVGL/lv_conf.h`
4. CCMRAM 64KB 已用于 LVGL 堆；主 SRAM 含 38.4KB 显示条带缓冲（驱动层）
